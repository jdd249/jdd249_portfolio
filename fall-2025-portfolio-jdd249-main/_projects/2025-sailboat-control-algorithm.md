## Overview

This project models an rc sailboat and implements a control algorithm to guide the path of the sailboat along a predetermined set of waypoints. I, along with three other group members, decided on this project both because of a shared interest in sailing and because we were curious to explore the implementation of a control model for a more complex, non-linear system. The scope of the project extends from the definition of various parameters involved in the control of the boat, the modelling of system and performance constraints, and then to control system implementation and animation in MATLAB.
